# Hotel Reservation Application 

## Overview
The goal of this project is to design an application that facilitates room reservations for clients of 5 hotels as a part of a hotel chain.
- Implement an application using a Database Management System (DBMS) such as MySQL.
- Provide SQL code that supports all application functionalities.



### Programming Languages
- SQL:used to create, update, and retrieve data for room availability, pricing, amenities, and customer information.
- HTML/CSS/JavaScript: It determines the structure and layout of web pages,structure, styling, and interactivity.
